# 天・地・人｜雙人配對頁神秘力量版實作計畫

## 專案定位

這個網站不是做成可愛算命工具，而是做成「一打開就像進入命盤能量場」的單頁式配對體驗。

使用者輸入兩人的血型與生日後，前端會自動換算星座，後端再把資料送進 Google Gemini，產出結構化的配對分析。整體風格以「高級命理 × AI 靈性科技感」為主軸。

## 已鎖定決策

- 頁面定位：配對首頁
- AI：Google Gemini
- UI 主題：天／地／人三層式神秘力量版
- 字體：`Noto Serif TC` 標題 + `Noto Sans TC` 內文
- 動效：高動效，但支援 `prefers-reduced-motion`
- 初版範圍：單頁配對，不含登入、分享、歷史紀錄、多語系

## 頁面結構

### Hero 區

- 標題：天地人正在校準你們的氣場
- 副標：不是算命，是天地人能量正在替你解碼命運
- 中央放發光命盤能量圓
- 背景有星點與淡命盤環

### 主互動區

- 左側主卡片放雙人輸入
- 每人一張命盤卡
- 欄位：
  - 名稱（可選）
  - 血型（A / B / AB / O）
  - 生日（Date Picker）
- 生日輸入後前端自動顯示星座
- 主按鈕文案：啟動天地人解碼

### 輔助資訊區

- 今日主運卡
- 三宮快捷卡：
  - 天｜事業運
  - 地｜財運
  - 人｜感情運

### 結果區

- 主命盤能量指數
- 四項分析卡
  - 靈魂頻率
  - 感情磁場
  - 溝通流動
  - 長期命勢
- AI 命運摘要

## 技術架構

### 前端

- Next.js 14 App Router
- React 18
- TypeScript
- Tailwind CSS

### 後端

- Next.js Route Handler：`POST /api/analyze`
- Google Gemini SDK：`@google/genai`

### 資料流

1. 使用者輸入兩人資料
2. 前端驗證欄位
3. 前端以生日換算星座
4. 呼叫 `/api/analyze`
5. 後端送資料給 Gemini
6. Gemini 回傳 JSON 分析結果
7. 前端渲染結果卡片與能量條

## AI 規格

### 請求資料

```ts
{
  personA: { name: string; bloodType: 'A' | 'B' | 'AB' | 'O'; birthday: string; },
  personB: { name: string; bloodType: 'A' | 'B' | 'AB' | 'O'; birthday: string; }
}
```

### 回傳格式

```ts
{
  overall_score: number,
  personality: { score: number, description: string },
  love: { score: number, description: string },
  communication: { score: number, description: string },
  future: { score: number, description: string },
  summary: string
}
```

### Prompt 原則

- 繁體中文
- 神秘、權威、精準
- 不可可愛化、搞笑化
- 僅輸出 JSON
- 每段描述適合直接渲染在卡片上

## 驗收標準

- 兩人生日完整時可正常送出分析
- 缺生日時按鈕不可送出或有清楚提示
- Gemini 正常回傳時，分數與摘要完整顯示
- Gemini 格式異常時，畫面能顯示錯誤訊息
- 深色背景下文字對比足夠
- 手機版不破版
- 高動效下仍保持流暢
- 使用者若偏好減少動作，動畫能自動降級
