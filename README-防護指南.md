# 🛡️ 首頁防護系統 - 快速啟動指南

## ⚡ 三步啟動（最簡單的方式）

### 方式1：Windows 雙擊啟動（推薦）✅

在項目文件夾中找到 `start-protected.bat`，**雙擊它**

✨ 就這麼簡單！系統會自動：
- 啟動 dev server
- 啟動 24/7 監控衛士
- 顯示防護提示

### 方式2：命令行啟動

打開終端，執行：
```bash
npm run dev:safe
```

---

## 📊 系統工作方式

```
你的應用 ← 監控 ← 首頁衛士 🛡️
         ↓ 若失敗
       自動恢復
```

### 每10秒檢查一次：
- ✅ 首頁可訪問？ → 繼續監控
- ❌ 無法訪問？ → 失敗計數 +1
- 失敗3次？ → 🚀 **自動恢復流程啟動**

### 自動恢復做什麼：
1. 清理衝突進程
2. 修復配置問題
3. 清理損壞文件
4. 重新啟動服務
5. 驗證首頁恢復
6. 回到監控

## ❓ 常見問題

### Q: 為什麼要啟用防護？
A: 防止首頁突然消失，自動恢復問題，24/7 監控

### Q: 會影響性能嗎？
A: 不會。只是每10秒檢查一次（開銷極小）

### Q: 可以關閉嗎？
A: 可以按 Ctrl+C，但**不建議關閉**

### Q: 如何調整檢查間隔？
A: 編輯 `scripts/homepage-guardian.js` 中的 `CHECK_INTERVAL`

### Q: 衛士出問題了怎麼辦？
A: 查看 `.guardian-log.txt`，或手動重啟：
```bash
npm run dev
```

## 📞 什麼時候需要手動介入？

幾乎永遠不需要！除非：
- ❌ 代碼有語法錯誤（需要修復代碼）
