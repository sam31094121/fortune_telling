## 操作規則

### 規則 1：說「開啟網站」= 幫我跑 dev server
當我說「開啟網站」「開網站」之類的話，請自動執行：
```bash
cd "C:\Users\DRAGON\Desktop\命理"
npm run dev
```
等伺服器啟動後告訴我可以開 http://localhost:3000

> 本機開發版（localhost）和線上版（Vercel）是兩回事，本機修改即時反映。

## 金鑰與環境變數

| 位置 | 用途 |
|------|------|
| `.env.local`（本機） | 本機開發用，金鑰填在這裡 |
| Vercel → Settings → Environment Variables | 線上版用的金鑰 |

金鑰名稱：`GEMINI_API_KEY`
取得位置：https://aistudio.google.com/apikey

> `.env.local` 不會上傳 GitHub（已加入 `.gitignore`）
